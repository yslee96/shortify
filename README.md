# shortify
 Text Summarization Service using chatGPT API

## Summary
chatGPT API를 활용하여 사용자가 입력한 텍스트를 사용자가 원하는 분량으로 요약한 결과를 제공하는 웹서비스를 개발하고 있습니다. 현재 요약된 결과를 화면에 표시하는 단계까지 개발을 완료했습니다. 추후 번역 및 사용된 토큰 수를 표시하는 기능을 추가할 예정입니다.
